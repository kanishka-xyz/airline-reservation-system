# Airline-Reservation-System
 A full-stack airline booking platform built using React.js, Node.js, Express, and MongoDB, where users can search flights, select seats, and generate downloadable ticket PDFs in real-time.

🎯 Project Overview

The Airline Reservation System allows users to:

Search available flights by source, destination, date, and time

Select a preferred seat category

Automatically assign seat numbers

Generate a downloadable ticket in PDF format

Authenticate using secure login and registration

Manage flight data via admin functionality

The system is built with a scalable architecture separating frontend and backend services.
